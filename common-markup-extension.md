## Common markup entensions
| markup extension | purpose |
|------------------|---------|
| `{Binding ...}` | data binding (with `BindingContext` or `Source` & `Path`) |
| `{StaticResource ...}` | XAML resources with `x:key` |
| `{x:Null}` | to set `null` |
| `{x:Type someClass}` | target type |
| `{x:Array ...}` | arrays in XAML |
